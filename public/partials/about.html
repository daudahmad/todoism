About Todo-ism!

Todo-ism is a simple but effective todo list app built using AngularJS, Node.js and BlueMix.

  1. Add/delete as many lists as you like.
  2. Add/delete todos. Mark them done as they get completed.
  3. Lightning fast search for existing lists and todos.
  4. Track different project todos by creating a todo list for each project.
  5. Managers and team leads track their team's todos. Just create a new list for each person.

Check out the code at: https://github.com/daudahmad/todoism
